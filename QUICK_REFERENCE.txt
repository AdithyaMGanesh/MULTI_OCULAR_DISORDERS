## 🚗 The Two Lanes

### Lane 1 (Specialist - Left Card)
- **Input:** 224×224 image
- **Model:** Fusion DR Detection
- **Output:** Diabetic Retinopathy diagnosis (5 classes)
- **Classes:** No DR, Mild, Moderate, Severe, Proliferative

### Lane 2 (Generalist - Right Card)
- **Input:** 128×128 image  
- **Model:** Lightweight Custom
- **Output:** Glaucoma/Cataract diagnosis (3 classes)
- **Classes:** Normal, Glaucoma, Cataract

---

  "lane_1_specialist": {

## ⚙️ Manual Start (Alternative)

```bash
# Terminal 1: Backend
python -m uvicorn src.api_dual_model:app \
  --host 0.0.0.0 --port 8001

# Terminal 2: Frontend
cd frontend
npm install
npm start
```

## 🆘 Troubleshooting

| Problem | Solution |
|---------|----------|
| Port 8001 in use | Kill process on that port |
| Connection refused | Ensure backend is running first |
| Blank frontend | Clear browser cache + refresh |
| CORS error | Verify backend port 8001 is running |
| Demo mode | Model files not found (it's OK!) |
